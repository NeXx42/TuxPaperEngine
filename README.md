Basic C# Avalonia application for interacting with https://github.com/Almamu/linux-wallpaperengine.

Intention is to allow this to work with my dual monitor setup, for seemless display ive use my forked version of the repo to handle uv offsets.
https://github.com/NeXx42/linux-wallpaperengine

```bash
git clone https://github.com/NeXx42/linux-wallpaperengine-gui.git --recursive
```

```bash
make publish
```
